Experiment code for "Attention Constraints and Learning in Categories"

Folder Contents:

Main experimental files:
- FORECAST-v.html: variance experiment
- FORECAST-s.html: size experiment
- FORECAST-t.html: time experiment
- FORECAST-m-h.html: memory experiment
- FORECAST-v-o.html: latent experiment

Auxiliary experimental files:
- Chart.js, chartjs-plugin-datalabels.js: library for charts
- confirmation_code_H2C.html: confirmation code page
- consent-forecast.html, consent-forecast-3.html: consent form page
- debrief.html: debrief page
- save_data.php: code for saving data

Auxiliary folders:
- jspsych-5.0.3: jspsych folder of plugins
- imgs: images used in experiments
- data: empty folder needed to be there to save data into
